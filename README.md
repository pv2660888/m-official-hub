# m-official-hub

## 项目简介

`m-official-hub` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为页面资源的集中存放与版本管理空间。

项目旨在提供一个结构清晰、易于维护的 HTML 页面集合，便于开发者或团队快速查阅、引用或分发页面内容。所有页面均保持独立，互不依赖。

## 目录说明

```
m-official-hub/
├── pages/          # 存放所有独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/         # 页面共用的静态资源（如 CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
├── README.md       # 本文件
└── LICENSE         # 仓库许可证
```

- **pages/**：核心目录，每个 HTML 文件代表一个独立页面，文件名应具有描述性。
- **assets/**：存放页面共享的样式、脚本或媒体文件，避免重复引用。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接在浏览器中打开，无需额外服务器环境。
- 页面之间无交叉引用，每个页面可单独更新或删除。
- 建议在提交新页面时，附带简要说明（可在 commit message 中标注）。
- 本仓库不追踪外部链接或第三方资源，所有内容应尽量自包含。

## 维护说明

- 欢迎提交 Pull Request 添加新页面或优化已有内容。
- 请保持目录结构整洁，不随意新增顶级文件夹。
- 若需更新共用资源（如样式文件），请注意兼容性，避免破坏现有页面。
- 仓库维护者会定期审核提交，确保页面质量与一致性。
